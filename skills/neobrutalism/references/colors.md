# Color Palette Reference

## Base Colors (Structure)

```css
#000000  /* Black - borders, shadows, text */
#FFFFFF  /* White - backgrounds, contrast */
```

## Usage Rules

1. **Limit to 2-3 accent colors** per design
2. **Avoid problematic pairings** (yellow + cyan) for accessibility
3. **Use full saturation** - no pastels or muted tones
4. **No gradients** - colors must be solid
5. **Maintain 4.5:1 contrast ratio** minimum
