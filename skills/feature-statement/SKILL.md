# Feature Statement Template

## Format

### 1. Feature Summary
**Nếu chúng ta làm** [Feature] **thì** [Who - ai] **sẽ** [Value - nhận được gì]

### 2. Evidence/Signal
Vì sao bạn nghĩ đây là cơ hội lớn?
- Data points
- User feedback
- Market signals

### 3. Feature Description
Mô tả chi tiết hơn về Feature:
- What: Feature làm gì
- How: Hoạt động như thế nào
- Scope: Phạm vi và giới hạn
